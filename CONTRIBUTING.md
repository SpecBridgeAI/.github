# Contributing

Thank you for your interest in SpecBridge AI.

Before opening a pull request, review the target repository's README and contribution instructions. Keep changes focused, explain the validation performed, and avoid including confidential or customer-derived material.

## Safety requirements

- Use synthetic or explicitly approved public data only.
- Never commit credentials, tokens, personal data or internal URLs.
- Preserve human-review and claim-limitation language.
- Report security issues privately according to [SECURITY.md](SECURITY.md).
- Do not add dependencies or external services without an explicit architecture and security review.

Repository maintainers may close contributions that fall outside the repository's published scope.
